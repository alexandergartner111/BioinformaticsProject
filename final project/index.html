Gene Annotation Comparison

The purpose of this website is to compare gene annotations between GRCh37.82 GTF from ENSEMBL and GRCh38.p5 GFF from NCBI of the species Homo sapien.

The homepage, which you are currently on, provides access to the comparison page where you can query our databse and compare genes and transcripts.

The Overview page lists summary information for the two genomes in comparison.

The Comparison page to query the database and see the comparison between the gff format from NCBI and gtf format from ENSEMBL.
